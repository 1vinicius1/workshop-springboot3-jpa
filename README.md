# Como executar o projeto

## Back end
Pré-requisitos: Java 17

```bash
# clonar repositório
git clone https://github.com/1vinicius1/workshop-springboot3-jpa

# entrar na pasta do projeto back end
cd backend

# executar o projeto
./mvnw spring-boot:run
```
